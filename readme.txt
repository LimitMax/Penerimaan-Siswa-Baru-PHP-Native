Program PSB Sederhana

Pembuatan menggunakan PHP, Framework Bootstrap, dan JQuery dari Bootstrap.

Fitur yang ada pada program ini:
#Admin
- Dapat melihat detail Pendaftar
- Dapat melakukan perubahan status pendaftaran (Diterima/Cadangan/Tidak Diterima)
- Dapat melihat detail akun pengguna


#Pendaftar
- Dapat melakukan registrasi akun baru
- Dapat melakukan pengisisan formulir pendaftaran
- Dapat melihat detail informasi pendaftar dan status pendaftaran
- Dapat melakukan cetak kartu bukti pendaftaran
